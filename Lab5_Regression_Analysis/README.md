# Lab 5: Regression Analysis

## Description
This lab focuses on applying regression techniques to predict continuous values. It uses the Boston Housing dataset to predict house prices based on various features.

## Topics Covered
- Introduction to Regression
- Linear Regression Model
- Dataset loading and preprocessing
- Feature selection
- Model training and prediction
- Model evaluation

## Tasks Performed
- Loaded the Boston Housing dataset
- Explored and analyzed the dataset
- Applied Linear Regression model
- Predicted house prices
- Evaluated model performance using metrics like Mean Squared Error (MSE)

## Files Included
- Lab5_Regression_Analysis.ipynb
- LAB5_BOSTON__housing_problem.ipynb

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Conclusion
This lab demonstrated how regression models can be used to predict continuous outcomes. It provided practical understanding of model training, prediction, and evaluation.
