# Lab 6: KNN Classification

## Description
This lab focuses on classification techniques using the K-Nearest Neighbors (KNN) algorithm. It demonstrates how to classify data points based on similarity with their nearest neighbors.

## Topics Covered
- Introduction to Classification
- K-Nearest Neighbors (KNN) Algorithm
- Dataset loading and preprocessing
- Feature scaling
- Model training and prediction
- Model evaluation

## Tasks Performed
- Loaded and explored the dataset
- Preprocessed the data for classification
- Applied KNN algorithm for classification
- Predicted class labels for test data
- Evaluated model performance using accuracy score

## Files Included
- Lab6_KNN_Classification.ipynb

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Conclusion
This lab provided practical understanding of classification techniques using KNN. It demonstrated how data can be classified based on similarity and distance metrics.
